# v0.1.2 Release Announcement

## Tweet 1 (hook + image)

agentjail v0.1.2 is out.

Your coding agent now has a local decision store, a secrets broker, and a web UI to see everything it tried to do.

Every tool call. Every deny. Every session. Queryable, filterable, replayable.

Thread 1/5

---

## Tweet 2 (what's new, the meat)

2/5 What shipped:

SQLite decision store. Every tool call your agent makes gets logged with the rule that fired, the action taken, and the latency. WAL mode, concurrent readers, automatic retention cleanup.

`agentjail replay --session <id>` plays back any session like a flight recorder.

---

## Tweet 3 (shield + secrets)

3/5 Shield got serious:

. Environment audit at launch (catches root, leaked creds, IMDS)
. Strips ambient credentials before the agent sees them
. agentjail-secrets broker issues scoped, short-lived creds on demand
. Landlock network rules on Linux 6.7+

The agent gets exactly the access it needs. Nothing more.

---

## Tweet 4 (web UI)

4/5 New: `agentjail ui`

Local web viewer backed by the SQLite store. Server-side filtered queries (action, tool, rule). Resizable panes. Agent logos for Claude, Cursor, Codex, OpenCode. Collapsible audit log.

See what your agent did, session by session, decision by decision.
